# Hotpoint_Appliances_Analytics

This project demonstrates **data analytics for Hotpoint appliances retail**, focusing on various well-known appliance brands sold under a Hotpoint-like catalog in Kenya.

**Objectives:**  
- Track **sales, revenue, and inventory levels per product**  
- Analyze **monthly sales trends**  
- Monitor **top-performing brands and products**  
- Provide insights for **inventory management, marketing strategy, and investor reporting**

**Skills demonstrated:** SQL, Sales & Revenue Analytics, Inventory Analytics, Product Performance

**Brands included:**  
- Von  
- Hisense  
- LG  
- Black+Decker  
- Samsung  
- Simfer  
- Dyson
